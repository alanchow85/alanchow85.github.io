What is the difference between supervised and unsupervised learning?

If we recalled a simple regression, Y = mX + C, the Y is the dependent variable and X is the independent variable. So for supervised learning, all the training data has the Y value. As for unsupervised, we can be trying to cluster some data into a certain number of bundle without have the need for Y value.
